# Tugas III Mobile Programming
- Silahkan download dan install sqlite di laptop masing-masing
- Buat 1 database dengan format “nim_nama.db”
- Buat 2 buah tabel:
  - Tbl_product berisi:
      - Id int
      - Sku text
      - Nama_produk text
      - Kategori text
  - Tbl_customer berisi
      - Id int
      - Nama_pelanggan text
      - Alamat char
      - Email varchar
